## üöÄ Quick Start

### Prerequisites

```bash
Node.js >= 16.x
npm >= 8.x
```

### Installation

```bash
# Clone the repository
git clone https://github.com/debarun1234/my-react-app.git
cd my-react-app

# Install dependencies
npm install

# Start development server
npm run dev
```

## üîß Configuration

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GITHUB_USERNAME=your_github_username
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### Customization

1. **Personal Info:** Update content in each component file (`src/components/`)
2. **Theme Colors:** Modify `src/theme/index.js`
3. **Certificates:** Add PDF files to `src/assets/` and update component imports
4. **Resume:** Replace `public/Debarun Ghosh_Resume.pdf` with your resume
